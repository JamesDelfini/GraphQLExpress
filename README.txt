# Initiate Application
    # From Scratch
        Initiate the package.json application information
        > npm init -y
    # Downloaded Direct to Github
        This will download all dependencies inside of package.json
        > npm install

        # Project Prerequisites
            # Express and GraphQL
                > npm i express express-graphql graphql

            # Nodemon
                > npm i --save-dev nodemon

        # Commands
            # To start the development environment of server-side express
                The commands of nodejs can be found in package.json
                > npm run dev:start
